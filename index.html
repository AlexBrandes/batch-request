Batch Request

Batch Request is a lightweight library for Node.js which allows clients to send multiple requests to a server running Node.js in batch.

We developed this at SocialRadar particularly for use with mobile devices connecting to Node.js. For mobile devices particularly, spinning the cell modem up is an expensive operation and includes a fair amount of latency (we found an average of around 300ms from an iPhone 5s on LTE!). So we needed a way to be able to send API requests in batch from these mobile devices to our API to minimize battery drainage and latency on the client.

Using Batch Request you can send a single request which represents many requests and receive the results of all those requests in a single reply.

Batch Request is open source software and is released under the developer and business-friendly MIT license.
