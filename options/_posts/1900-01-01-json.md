---
title: json
---

Boolean, will specify that the included `body` is JSON and adds a header of
`Content-type: application/json`. This is mostly a convenience method.

{% highlight json %}
{
    "myRequest1": {
        "json": true,
        "method": "POST",
        "uri": "http://api.mysite.com/users/1",
        "body": {
            "first_name": "Victor"
        }
    }
}
{% endhighlight %}
