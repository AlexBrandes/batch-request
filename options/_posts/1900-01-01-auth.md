---
title: auth
---

Send basic auth information with this request. Expects an object with `user`
and `pass` keys.

{% highlight json %}
{
    "myRequest1": {
        "auth": {
            "user": "victor",
            "pass": "mysecurepassword"
        },
        "method": "GET",
        "uri": "http://api.mysite.com/users/1"
    }
}
{% endhighlight %}

*default: none*
